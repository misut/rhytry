# 프로젝트: rhytry

## 1. 개요

`rhytry`는 Rust 언어와 Bevy 게임 엔진을 사용하여 개발하는 리듬 게임입니다. Bevy의 ECS(Entity-Component-System) 아키텍처와 플러그인(Plugin), 상태(State) 기반으로 코드가 구성됩니다.

## 2. 주요 기술 및 도구

*   **언어**: Rust
*   **게임 엔진**: Bevy
*   **에셋 로더**: `bevy_asset_loader`

## 3. 주요 게임 개념

*   **`Onpu`**: 게임 내 음표를 나타내는 컴포넌트.
